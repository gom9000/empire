Empire - a java utility library
Empire is a java library of custom hierarchy tag and general-purpose utility classes.
Empire modules implements facilities to data and datatype conversions, configuration, logging, filesystem manager and more over.
Empire architectural model leads to developing code compliant with modern j2ee principles.
Empire rises from Gos-Work's ash.
Empire, state: building!


Features
The Empire library works with jre version 1.4.

The binary and source dist is available at: http://www.gommagomma.net/gos95/empire.
